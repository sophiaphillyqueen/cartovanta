# cartovanta
Basic tools of the -cartovanta- card system

## Installation
For a standard install on your user account, first make sure that either `~/local/bin` or `~/bin` is on your `PATH`.
Then, go to the top level of the unpacked source tree and run the command:
```
perl install.pl
```

For instructions for systemwide installs, as well as other variations of installation, view the installation helpfile (also from the top level of the unpacked source tree) by running:
```
perl install.pl --help
```
